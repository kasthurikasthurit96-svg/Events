# JavaScript Events Demo

## Project Overview

This project demonstrates different JavaScript events using HTML, CSS, and JavaScript.
The webpage is interactive and responds to user actions like mouse clicks, keyboard presses, mouse hover, focus events, and selection changes.

# Events Used in the Project

## 1. Click Event (`onclick`)

Triggers when the user clicks the button.

## 2. Double Click Event (`ondblclick`)

Triggers when the user double-clicks the button.

## 3. Mouse Over Event (`onmouseover`)

Changes the box color when the mouse moves over it.

## 4. Mouse Out Event (`onmouseout`)

Restores the original box color when the mouse leaves the box.

## 5. Key Press Event (`onkeypress`)

Displays the key pressed by the user.

## 6. Focus Event (`onfocus`)

Activates when the input field receives focus.

## 7. Change Event (`onchange`)

Triggers when the user selects a different option from the dropdown menu.
